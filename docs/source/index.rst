Learned Stackについて
-----------------------

| sudo-roaの学習の積み重ねをまとめたWebサイト

   * その日勉強した内容
   * メモのようなもの
- コンテンツ
   * 実際に勉強した内容を記述した項目群
   * ある程度まとめたもの
- 管理事項
   * Learned Stackを管理していく上で必要な事項を列挙した項目群
- 検索機能
   * 検索および索引機能

| 検索機能があるので過去の学習を参照しやすい点が採用理由
